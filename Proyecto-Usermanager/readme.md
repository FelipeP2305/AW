Proyecto 3 --- Usermanager
AW

**Importante!!**

Para poder usar este proyecto deberas tener instalado lo siguiente:
- MySQL o MariaDB
- Un navegador web de tu preferencia
- Servidor Apache2 con sus extensiones de MySQL para una conexión correcta

**Prerrequisitos**

1) Primero copia la carpeta "Proyecto-Usermanager"
2) Debes tener creada la base de datos mediante las instrucciones en "sql/users.sql"
3) Para conectar con la base de datos creada, debes que configurar el archivo ubicado en "config/db.php", colocando su usuario, contraseña y nombre de la base de datos

**Modo de Uso**

Para usar esta aplicación deberas crear el usuario desde linea de comandos con el uso de MariaDB o MySQL

(Este mismo será el que gestioné el resto de usuarios

Acceder a la app mediante esta dirección web: http://localhost/AW/Proyecto-Usermanager/

Iniciar sesión con el usuario registrado en base de datos (dependiendo de tu rol podrás ver el CRUD)
